<div align="center">

### Installation

<pre>
curl -s https://raw.githubusercontent.com/x3ric/xtool/refs/heads/main/install | bash
</pre>

### Usage

<pre>
xtool [command] [args...]
</pre>

### Examples

<pre>
xtool click 1
xtool key Return
xtool mousemove 100 100
xtool type "Hello, World!"
</pre>
